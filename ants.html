Ants Simulation.

A Python project that simulates the natural behavior of ants. For the visualization I used the matplotlib library. Due to the in red in yellow shown pheromone traces, the calculation quickly becomes very computationally intensive. To speed this up I implemented my vision of a quadtree in Python. A quadtree is a data structure that is used in 2D space to limit the number of points that need to be considered for a collision detection. With this I was able to reduce the time for a render with 1000 frames and 100 ants from about one hour to about one minute.

The code to the Project can be found here.

Ants start out walking around completely randomly, leaving behind 'home' pheromones. When an ant accidentally encounters a food source, it picks up food. It then looks to see if there are any 'home' pheromones near it. If so, the ant is more likely to turn in the direction of the highest pheromones concentration (not 100%, but much lower). Now it leaves behind 'food' pheromones. If another ant meets a food trail by chance, the probability is higher that it turns in the direction of the highest food pheromones than that it chooses the direction randomly (again not 100%).
